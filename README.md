## Description
Interactive mini application developed with HTML, CSS, and JavaScript.

## Technical Specifications

### Technologies Used
- **HTML5** - Semantic structure
- **CSS3** - Styling and responsive design
- **JavaScript (Vanilla)** - Interactive logic

### Project Structure
```
js-mini-app-1/
├── index.html              # Main page
├── scrip.js                # Main script
├── style.css               # Global styles
├── board.css               # Board-specific styles
├── README.md               # This file
├── pages/
│   └── game-normal-mode.html  # Game page in normal mode
└── images/
    └── board-images/       # Board images
```

### Key Features
- Interactive user interface
- Normal game mode
- Visual board system
- Responsive design

### Supported Browsers
- Chrome (version 90+)
- Firefox (version 88+)
- Safari (version 14+)
- Edge (version 90+)

### How to Use
1. Open `index.html` in a web browser
2. Interact with the application according to on-screen instructions
3. Access the game mode through available links

## Notes
- The `scrip.js` file contains the main logic (note: consider reviewing the filename spelling)
- Styles are distributed between `style.css` and `board.css`
- Images are organized in the `images/` folder
